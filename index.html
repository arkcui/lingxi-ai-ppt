第一页

🔥 浙大校招复盘：得与失

✅ 得之所在（What Worked）

  • 时代叙事打动人
    不讲岗位讲机会 → 学生感知到"参与时代级变革"
  • 热爱成为筛选器
    真玩家识别真玩家 → 当天发出5个Offer
  • 愿景张力足
    AI原生团队不是补位，是搭建新东西

⚠️ 失之改进（What to Improve）

  • 后续承接速度
    现场热度高，但24小时内建联效率可提升
  • 材料标准化与个性化的平衡
    保留业务判断，但需更结构化可复制
  • 校招知识库沉淀
    每场固定记录：Top5问题、最打动3个表达
第二页

🎯 两年AI转型目标与节奏

核心检验指标（三大支柱）

🏆 支柱一：AI原生应用
规模与影响力 — 从0到1打造行业影响力AI产品
💼 支柱二：前台业务
经营与市场竞争力 — 九游、交易猫、biubiu AI赋能增长
⚡ 支柱三：组织迭代
战斗力与效率 — 细胞式组织、数字员工、人机协同

⏰ 节奏规划

Phase 1（2026 H1）
AI工具渗透全员 → 数字员工MVP → 细胞组织试点
Phase 2（2026 H2）
AI原生产品1.0 → 前台业务AI化 → 组织形态升级
Phase 3（2027）
规模化与影响力 → 行业标杆确立 → ASI路径清晰

第四页

⚡ 组织形态升级的探索目标

🎯 目标一：数字员工计划

基于AI时代光型人才要求，由中台联手共同开发数字员工计划

  • 为在岗及新人提供即插即用的咨询、分析类服务Agent群组
  • 降低专业门槛，提升全员AI素养与工作效率

🚀 目标二：一人即组织

朝着"一人即组织"的形态演进

  • 鼓励光型人才打开想象力
  • 开发更多游戏垂类AI原生应用
  • 个体创造力 × AI工具 = 无限可能

💡 目标三：预期效果

颠覆性改变产品开发流程：

想法 → Demo → MVA 版本的时间和成本大幅下降

中台动态支持机制：

  • 技术中台：根据产品阶段评估技术可行性
  • 市场中台：匹配推广资源与获客策略
  • 用增中台：配置增长资源与运营支持
